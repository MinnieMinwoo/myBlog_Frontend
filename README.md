# myBlog_Frontend

Make my own blog project.

## features

-   Login
    -   Email Login
-   Create and Read posts using markdown

## project stack

-   React
-   React Router
-   Typescript
-   Recoil
-   styled-components
-   react-md-editor
-   Jest
-   React Testing Library
-   Firebase
    -   Authentication
    -   Firestore Database
-   Github Pages
